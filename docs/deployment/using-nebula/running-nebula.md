description: How to run Nebula
---

Once we wrote our `lyra.yml` and `get-data.js` file, we can finally run Nebula.

Running it is as simple as:

```bash
$ nebula bundle
```

It will produce a `dist/index.js` file, which we can deploy to Cloudflare
Workers.
